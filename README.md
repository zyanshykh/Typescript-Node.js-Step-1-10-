# Typescript Node.js Step-by-Step Guide (Step 1-10)

Yeh project Node.js aur TypeScript ke basic se lekar advanced concepts ko step-by-step sikhane ke liye banaya gaya hai. Ismein har concept ke liye alag file maujood hai taake samajhne mein aasani ho.

---

## 🚀 Features & Steps Covered

*   **Step 01: Strong Typing** – Variables aur data types ko strictly define karna.
*   **Step 02: Const & Let** – Scope management aur variable declarations.
*   **Step 03: Modules & Packages** – Native ECMAScript modules, Inquirer CLI inputs, aur Chalk library ka istemal.
*   **Step 04: Unions & Literals** – Multiple types aur specific custom values assign karna.
*   **Step 05: Advanced Objects** – Structural typing, nested objects, intersection types, aur `any`/`unknown`/`never` types ka mukammal istemal.

---

## 🛠️ Installation & Setup

Project ko apne local system par chalane ke liye niche diye gaye steps ko follow karein:

1. Repository ko clone karein:
```bash
git clone [https://github.com/zyanshykh/Typescript-Node.js-Step-1-10-.git](https://github.com/zyanshykh/Typescript-Node.js-Step-1-10-.git)
cd Typescript-Node.js-Step-1-10-
npm install
npx ts-node step01_strong_typing.ts
```
